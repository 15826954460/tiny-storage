#### 镜像源
- 海外私有源：http://npm.4wps.net
- npm源：https://registry.npmjs.org/

#### npm、yarn查看和更换镜像源
- npm/yarn config get registry // 查看当前镜像源
- npm/yarn config set registry https://registry.npmjs.org // 设置npm镜像源
- npm config set registry http://npm.4wps.net // 设置海外镜像源
